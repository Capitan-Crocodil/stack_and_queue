# Реализация стека и очереди на C++

## Описание
Данная программа демонстрирует работу двух структур данных:
- **Стек (Stack)** - реализован на основе односвязного списка
- **Очередь (Queue)** - также реализована на основе односвязного списка

## Функциональность

### Класс Stack
- `spush(value)` - добавление элемента на вершину стека
- `spop()` - извлечение элемента с вершины стека
- `speek()` - просмотр верхнего элемента без удаления
- `sempty()` - проверка стека на пустоту
- `sprint()` - вывод всех элементов стека

### Класс Queue
- `qpush(value)` - добавление элемента в конец очереди
- `qpop()` - извлечение элемента из начала очереди
- `qpeek()` - просмотр первого элемента без удаления
- `qempty()` - проверка очереди на пустоту
- `qprint()` - вывод всех элементов очереди

## Пример работы
Программа выполняет следующие действия:
1. Заполняет стек s1 и очередь q1 числами от -20 до 60
2. Формирует стек s2 из чисел, **не кратных 7**
3. Формирует очередь q2 из чисел, **кратных 7**
4. Выводит результаты на экран

## Запуск программы
```bash
g++ main.cpp -o program
./program
```
